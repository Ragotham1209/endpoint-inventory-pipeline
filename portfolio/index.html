Enterprise Endpoint Inventory Pipeline

SCD Type 2 Pipeline on Azure Databricks

A production-grade data engineering pipeline that collects system inventory from 1M+ Windows endpoints, ingests payloads through auto-scaling Azure Functions (EP1), and maintains full dimensional history using SCD Type 2 merge patterns on Databricks Delta Lake. The architecture features VNet-injected compute, Unity Catalog governance, SHA-256 change detection at the edge, and Medallion Architecture processing with automated quarantine isolation.
